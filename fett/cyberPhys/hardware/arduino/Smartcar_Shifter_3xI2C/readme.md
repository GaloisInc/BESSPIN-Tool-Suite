# Teensy 3.6 I2C Interfaces
## I2C0
- Address = 0x30
- SCL = 100 kHz

## I2C1
- Address = 0x31
- SCL = 100 kHz

## I2C2
- Address = 0x32
- SCL = 100 kHz

_I2C interface parameters are reconfigurable in Arduino sketch_

# Reported Gear States
- PARK = 0x28
- REVERSE = 0x27
- NEUTRAL = 0x26
- DRIVE = 0x25
- MANUAL DRIVE = 0x23
- MANUAL PLUS = 0x29
- MANUAL MINUS = 0x2A
